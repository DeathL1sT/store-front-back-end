# Store Front Back-End.

## Installation

Install deps:

```bash
npm install
```

Run for development

```bash
npm run dev
```

Build for Prodcution:

```bash
npm run build
```

Run Tests:

```bash
npm run test
```

Formatting and Linting:

```bash
npm run format
npm run lint
```
